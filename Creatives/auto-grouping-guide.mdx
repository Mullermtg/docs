Auto-grouping lets Adtool combine creative files that belong together into one creative group when their names and placement tags match. Use it when the same creative concept has separate files for different placements, such as feed, story, reel, or horizontal formats.

## What must match

Adtool checks these requirements before grouping creatives automatically.

| **Requirement** | **What Adtool checks** |
| --- | --- |
| File naming | The files use matching creative names after the placement tag is removed. |
| Placement tags | Each file includes a supported placement tag, such as `1x1`, `4x5`, `9x16`, or `16x9`. |
| Same concept or base name | The remaining name points to the same creative concept. |
| Different placement sizes | The files are made for different supported placements. |
| Same file type | The files use the same file type. |

If Adtool is unsure whether files belong together, they are not grouped automatically.

## Supported placement tags

Add a placement tag to the file name so Adtool can identify which placement each creative belongs to.

| **Placement** | **Accepted tags** |
| --- | --- |
| Feed placement | **Square:**`1x1`, `1:1`, `1_1`, `1-1`, `1.1`, `(11)`, `1080x1080`, `1200x1200`, `square`, `feed`, `post`, `ig`, or a clear square pixel size like `1440x1440`.<br /><br />**Portrait:**`4x5`, `4:5`, `4_5`, `4-5`, `4.5`, `(45)`, `1080x1350`, or a clear 4:5 pixel size like `1080x1350`. |
| Story / Reel | `9x16`, `9:16`, `9_16`, `9-16`, `9.16`, `(916)`, `1080x1920`, `story`, `stories`, `reel`, `reels`, `vertical`, or a clear 9:16 pixel size like `1440x2560`. |
| Horizontal | **Landscape:**`16x9`, `16:9`, `16_9`, `16-9`, `16.9`, `(169)`, `1920x1080`, `landscape`, or a clear 16:9 pixel size like `1920x1080`.<br /><br />**Wide:**`1.91:1`, `1.91`, `1:91`, `1_91_1`, `1-91-1`, `(1911)`, `191x100`, `1200x628`, `horizontal`, `wide`, or a clear 1.91:1 pixel size like `1200x628`. |

Pixel sizes can use `x`, `X`, or `×`, for example `1440x2560`, `1440X2560`, or `1440×2560`.

## How placement tags should be written

The placement tag must stand on its own in the file name. It can be separated with a space, underscore, hyphen, or brackets.

```text
Summer Sale 9x16.jpg
Summer Sale_9x16.jpg
Summer Sale-9x16.jpg
Summer Sale (916).jpg
```

The tag should not be hidden inside another word or number. For example, `campaign1.1test.jpg` is not treated as a square creative because the `1.1` tag is part of a longer word.

## How names are matched

Adtool removes the placement tag from the file name, then compares the remaining name.

| **File name** | **Name Adtool compares** |
| --- | --- |
| `PA26-9.16-Benilda.jpg` | `pa26_benilda` |
| `PA26-1.1-Benilda.jpg` | `pa26_benilda` |

Because the remaining name is the same, these creatives can be grouped together.

## Examples that work

These file names can be used for auto-grouping:

```text
Summer Sale 1x1.jpg
Summer Sale_9x16.mp4
Summer Sale-horizontal.png
Summer Sale (916).jpg
PA26-1.1-Benilda.jpg
```

`Summer Sale (916).jpg` works because bracket tags are accepted. `PA26-1.1-Benilda.jpg` works because the tag is separated by hyphens.

## Examples that do not work

These file names are not grouped through normal placement tag matching.

| **File name** | **Why it does not work** |
| --- | --- |
| `campaign1.1test.jpg` | The placement tag is hidden inside another word. |
| `case-9.16test.jpg` | The placement tag is not separated from the rest of the name. |
| `DSCO1945.jpg` | There is no placement tag. |
| `creative-1024x768.jpg` | The size is not a supported placement. |
| `Summer Sale.jpg` | There is no placement tag. |

## Backup rule

If the full file names are exactly the same, Adtool can still group creatives by checking the actual creative dimensions.

| **File name** | **Actual size** |
| --- | --- |
| `Summer Sale.jpg` | 1:1 |
| `Summer Sale.jpg` | 9:16 |

These can be grouped because the full file name is the same and the actual sizes are different supported placements.

The backup rule does not work when the full file names are different.

| **File names** | **Why it does not work** |
| --- | --- |
| `Summer Sale.jpg`<br />`Summer Sale.png` | The full file names are different because the file types are different. |

## When creatives are not grouped

Adtool will not auto-group creatives when:

1. The creative sizes are the same.
2. The files use different file types.
3. The actual size is not supported.
4. The placement tag is missing.
5. The placement tag is hidden inside another word.
6. Adtool is unsure which files belong together.

## Group size

An auto-group can contain 2 or 3 creatives. It cannot contain more than 3 creatives.
