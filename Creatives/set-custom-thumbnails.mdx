Custom thumbnails control the preview image shown for video creatives before you launch them. You can either choose a frame from the video or upload a separate thumbnail image.

## Step-by-step guide

1. Go to the **Creatives** tab.

<img
  src="/images/Custom_thumbnail_1.png"
  alt="Creatives tab in Adtool"
  title="Creatives tab"
  style={{ width:"100%" }}
/>

2. Select the videos you want to update thumbnails for.

<img
  src="/images/Custom_thumbnail_2.png"
  alt="Selected video creatives in the creative library"
  title="Selected video creatives"
  style={{ width:"100%" }}
/>

3. Click the thumbnail icon in the top-right corner.

<img
  src="/images/Custom_thumbnail_3.png"
  alt="Thumbnail action button in the creative library"
  title="Thumbnail action icon"
  style={{ width:"100%" }}
/>

4. Choose whether to select a frame from the video or upload a custom thumbnail.

<img
  src="/images/Custom_thumbnail_4.png"
  alt="Thumbnail selection options"
  title="Custom thumbnail options"
  style={{ width:"100%" }}
/>

5. Click **Confirm** to update the thumbnail.

<img
  src="/images/Custom_thumbnail_5.png"
  alt="Confirm custom thumbnail update"
  title="Confirm custom thumbnail"
  style={{ width:"100%" }}
/>

## Thumbnail options

You can update video thumbnails in two ways:

- **Choose a frame from the video:** Select one of the frames from the video and use it as the thumbnail.
- **Upload a custom thumbnail:** Upload a separate image file and use it as the thumbnail for the selected video creative.

## Recommendation

If you upload a custom thumbnail, use the first frame of the video when possible. This keeps the preview consistent with the video and avoids manually creating a separate thumbnail for each video.
