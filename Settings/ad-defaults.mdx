Use ad defaults to make every new launch batch start with the settings your team normally uses. This helps keep launches consistent by pre-filling the website link, selecting the preferred CTA when possible, and deciding whether ads should launch as active or inactive by default.

Ad defaults are useful when your team normally sends traffic to the same website, uses the same call to action, or wants new ads to launch in a predictable status.

## What ad defaults control

| **Default** | **What it does** |
| --- | --- |
| Default website link | Adds the website link Adtool should use when creating new ads. |
| Default CTA | Selects the preferred call to action when that CTA is available for the ad setup. |
| Default ad status | Sets whether newly launched ads should be active or inactive by default. |

## When to use ad defaults

Use ad defaults when:

- Most ads should send traffic to the same website.
- Your team normally uses one preferred CTA.
- You want ads to launch as inactive first, so they can be reviewed before going live.
- You want ads to launch as active by default, so approved batches can start running immediately after launch.

## Step-by-step guide

1. Go to Settings in the left navigation bar.
2. Select the default settings or ad account settings you want to adjust.
3. Find the ad defaults section.
4. Add the default website link you want Adtool to use.
5. Select the default CTA you want Adtool to apply when available.
6. Choose whether ads should launch as active or inactive by default.
7. Save your changes.

## How each default works

### Default website link

The default website link is the website link Adtool should use when creating new ads.

### Default CTA

The default CTA is the preferred call to action Adtool should select when that CTA is available for the ad setup.

### Default ad status

The default ad status controls whether newly launched ads should be active or inactive by default.

## Recommendation

If your team reviews ads before they go live, set the default ad status to inactive. If your workflow is built around launching approved batches immediately, set the default ad status to active.

For the website link and CTA, choose the values your team uses most often. This keeps launch setup faster while still allowing batch-specific changes when needed.
