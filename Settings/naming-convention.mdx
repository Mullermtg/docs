Use a standard naming convention to control how Adtool builds ad names when ads are created. Naming conventions can be adjusted at two levels: across the organization through `Default settings`, or for a selected ad account when one account needs its own naming structure.

Use this page when you want launched ads to follow a predictable naming structure across batches.

## Step-by-step guide

<Steps>
  <Step title="Go to Settings" titleSize="h4">
    Go to `Settings`.

    <img
      src="/images/FB_IG_PAGE1.png"
      alt="Settings page in Adtool"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Choose where the naming convention should apply" titleSize="h4">
    Select `Default settings` to adjust the organization-wide naming convention, or select an ad account to adjust the naming convention for a specific ad account.

    <Frame>
      ![Settings Select Ad Account](/images/settings_select_ad-account.png)
    </Frame>
  </Step>
  <Step title="Find Naming Convention" iconType="solid" titleSize="h4">
    Scroll to the `Naming Convention` section.

    <Frame>
      ![Setting Naming Convention in settings](/images/settings_naming_convention.png)
    </Frame>
  </Step>
  <Step title="Build the naming convention" titleSize="h4">
    Add the fields, field order, separators, and custom labels or values you want Adtool to use when creating ad names.
  </Step>
</Steps>

## What you can set in a naming convention

| **Option** | **What it controls** |
| --- | --- |
| Fields | The values included in the ad name, such as filename, ad type, date, and custom fields. |
| Order | The sequence of fields in the final ad name. |
| Separators | The format placed between fields in the final ad name. |
| Custom labels and values | Static text or configurable labels that should appear in the ad name exactly as you define them. |

## Naming fields

<Expandable title="Filename">
  Use `Filename` to add the uploaded creative file name to the ad name. This is useful when each creative needs a dynamic identifier in the final ad name.

  If your filenames are structured well, `Filename` can make ad names highly specific and easier to distinguish across a batch.

  | **Creative filename** | **Ad name** |
  | --- | --- |
  | launch\_ads\_fast-9x16.mp4 | 10/28 \| VID \| Funnel \| launch\_ads\_fast |
</Expandable>

<Expandable title="Ad type">
  Use `Ad type` to identify the type of ad Adtool is creating. It can identify image ads, video ads, flex ads, carousel ads, partnership ads, and organic ads.

  You can customize what each ad type writes into the ad name. For example, you can use `img` for image ads, `vid` for video ads, or another label that matches your team's naming style.
</Expandable>

<Expandable title="Date">
  Use `Date` to add the date when the ad is created in Adtool. You can choose from several date formats so the date matches your desired naming style.
</Expandable>

<Expandable title="Custom Field">
  Use `Custom Field` for static text that should appear exactly as entered in every ad name.

  You can also use a `Custom Field` as a placeholder. For example, if you need country in your ad names, you can add a country placeholder and later adjust it with the `Find & Replace` function in `Launch`.
</Expandable>

<Expandable title="Separators">
  Separators control what appears between fields in the final ad name. Select the separator format your team already uses in its naming system.
</Expandable>

## Before you save

Check these items before saving the naming convention:

| **Check** | **Why it matters** |
| --- | --- |
| Correct settings level selected | `Default settings` changes the organization-wide naming convention. A selected ad account changes only that account's naming convention. |
| Naming fields are in the right order | Ad names are built from the fields in the order you set. |
| Filename behavior is intentional | Structured filenames can make each ad name more specific across a batch. Messy filenames can make ad names harder to scan. |
| Ad type labels match your system | Custom ad type labels let you decide what image, video, flex, carousel, partnership, and organic ads write into the ad name. |
| Custom fields are final or intentional placeholders | Static custom fields appear exactly as entered unless changed later. |

## Examples of naming conventions

| **Naming convention** | **Example name** |

## Common cases

<Expandable title="I want the same naming convention across the organization">
  Select `Default settings`, review the naming fields, order, separators, and custom labels, then save. This sets the organization-wide naming convention.
</Expandable>

<Expandable title="I want one ad account to use a different naming convention">
  Select the ad account, adjust the naming fields, order, separators, and custom labels, then save. This changes the naming convention for that selected ad account.
</Expandable>

<Expandable title="I need each creative or ad to be easy to identify">
  Add `Filename` to the naming convention. Structured filenames can make each ad name more specific and easier to distinguish across a batch.
</Expandable>

<Expandable title="I need ad type labels that match my team's system">
  Add `Ad type`, then customize the labels Adtool writes for image ads, video ads, flex ads, carousel ads, partnership ads, and organic ads.
</Expandable>

<Expandable title="I need to change one part of names during launch">
  Use a `Custom Field` as a placeholder, then adjust the value with `Find & Replace` in `Launch` when you prepare the batch.
</Expandable>

<Expandable title="My ad names look too long">
  Use fewer fields, shorten custom field values, or clean up creative filenames before adding the creatives to Adtool.
</Expandable>
