Use a standard naming convention to define how Adtool should build ad names by default for an ad account. Use this page when you want launched ads to follow a predictable naming structure across batches.

## What this setting affects

| **Setting** | **What it affects** |
| --- | --- |
| Selected ad account | The default ad naming convention for that account. |
| Naming fields | The parts Adtool uses to build each ad name, such as date, file type, file name, and custom fields. |
| Separators | The characters placed between naming fields. |
| `Use for all accounts` | Applies the same naming convention across your accounts and replaces account-specific naming defaults. |

## Naming fields

<Expandable title="File Type">
  Use `File Type` to add a label for whether the creative is an image or a video. You can set custom labels for image and video file types in your naming convention.
</Expandable>

<Expandable title="File Name">
  Use `File Name` to add the creative file name to the ad name.

  Example:

  ```text
  Creative name: launch_ads_fast-9x16.mp4
  Ad name: 10/28 | VID | Funnel | launch_ads_fast
  ```

  For cleaner ad names, adjust the concept description or file name before adding the creative to Adtool.
</Expandable>

<Expandable title="Date">
  Use `Date` to add the date when the ad is created within Adtool.

  Available date formats include:

  | **Format** | **Example** |
  | --- | --- |
  | `DD/MM` | `28/10` |
  | `MM/DD` | `10/28` |
  | `YYYY-DD-MM` | `2025-28-10` |
  | `YYYY-MM-DD` | `2025-10-28` |
  | `DD-MM-YYYY` | `28-10-2025` |
  | `MM-DD-YYYY` | `10-28-2025` |
  | `MMM DD` | `Oct 28` |
  | `DD MMM` | `28 Oct` |
  | `Week` | `Week 44` |
</Expandable>

<Expandable title="Custom Field">
  Use `Custom Field` for static text that should appear exactly as entered in every ad name.

  You can also use a `Custom Field` as a placeholder. For example, if you need country in your ad names, you can add a country placeholder and later adjust it with the `Find & Replace` function in `Launch`.
</Expandable>

<Expandable title="Separators">
  Choose the separator placed between naming fields.

  | **Separator** | **How it appears** |
  | --- | --- |
  | Pipe | Vertical bar between fields |
  | Dash | Dash between fields |
  | Underscore | Underscore between fields |
  | Space | Blank space between fields |
  | Comma | Comma between fields |
  | Dot | Dot between fields |
  | Tilde | Tilde between fields |
  | Double colon | Double colon between fields |
</Expandable>

## Before you save

Check these items before saving the default:

| **Check** | **Why it matters** |
| --- | --- |
| Correct ad account selected | The naming convention applies to the selected account unless `Use for all accounts` is enabled. |
| Naming fields are in the right order | Ad names are built from the fields in the order you set. |
| Custom fields are final or intentional placeholders | Static custom fields appear exactly as entered unless changed later. |
| `Use for all accounts` is intentional | It replaces custom naming conventions already set on other accounts. |

## Examples of naming conventions

**Example 1**

```text
Naming convention: Date (MM/DD) | CF | File Type (IMG/VID) | File Name
Ad name example: 10/28 | US | Awareness | VID | Hook 1 - Christian Dancing - 20% off focus
```

**Example 2**

```text
Naming convention: Date (MM/DD) | CF | CF | File Type (IMG/VID) | File Name
Ad name example: 10/28 | US | Awareness | VID | Hook 1 - Christian Dancing - 20% off focus
```

**Example 3**

```text
Naming convention: Week_CF_CF_File Type (IMG/VID)_File Name
Ad name example: Week 44_DK_Sales_High-heels_VID_hook1-UGC-Walking-down-the-street
```

## Common cases

<Expandable title="I want the same naming convention on every account">
  Enable `Use for all accounts`, review the naming convention, and save. This applies the same default across accounts and overrides account-specific naming conventions.
</Expandable>

<Expandable title="I need to change one part of names during launch">
  Use a `Custom Field` as a placeholder, then adjust the value with `Find & Replace` in `Launch` when you prepare the batch.
</Expandable>

<Expandable title="My ad names look too long">
  Use fewer fields, shorten custom field values, or clean up creative file names before adding the creatives to Adtool.
</Expandable>
