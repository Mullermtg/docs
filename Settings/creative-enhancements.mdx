Use default Creative enhancements to control which enhancement settings Adtool starts with when you create launch batches. Defaults can be set at three layers: organization-wide, ad account-wide, and batch-level.

## How the default layers work

| **Layer** | **Where it is set** | **What it affects** |
| --- | --- | --- |
| Organization-wide | `Settings` → `Default settings` → `Creative enhancements` | Applies the default to all current and future ad accounts in the organization. |
| Ad account-wide | `Settings` → select an ad account → `Creative enhancements` | Can override the organization-wide default and create a new default for that specific ad account. |
| Batch-level | `Launch` → `Additional settings` while creating a batch | Changes Creative enhancements only for the batch being created. It does not change saved organization or ad account defaults. |

<Note>
  Settings saved at the organization or ad account layer are defaults for future work. Batch-level changes in `Launch` only apply to the batch you are creating at that moment.
</Note>

## Set the organization-wide default

Use the organization-wide default when every current and future ad account in the organization should start with the same Creative enhancements setup.

<Steps>
  <Step title="Open Settings" titleSize="h4">
    Go to `Settings` in the left navigation bar.

    <img
      src="/images/FB_IG_PAGE1.png"
      alt="Settings page in Adtool"
      title="FB IG PAGE1 Pn"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Open Default settings" titleSize="h4">
    Click `Default settings`.
  </Step>
  <Step title="Find Creative enhancements" titleSize="h4">
    Scroll to `Creative enhancements`.
  </Step>
  <Step title="Set the organization default" titleSize="h4">
    Change the Creative enhancements setting. When changed here, the default applies to all current and future ad accounts in the organization.
  </Step>
</Steps>

## Set the ad account-wide default

Use the ad account-wide default when one ad account should use a different Creative enhancements setup than the organization-wide default.

<Steps>
  <Step title="Open Settings" titleSize="h4">
    Go to `Settings` in the left navigation bar.
  </Step>
  <Step title="Select the ad account" titleSize="h4">
    Select the ad account you want to change.

    <img
      src="/images/Select_ad_accounts.png"
      alt="Ad account selector in Settings"
      title="Select Ad Accounts Pn"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Find Creative enhancements" titleSize="h4">
    Scroll to `Creative enhancements`.

    <img
      src="/images/select_creative-enhacement.png"
      alt="Creative enhancements section in Settings"
      title="Select Creative Enhacement Pn"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Set the ad account default" titleSize="h4">
    Change the Creative enhancements setting. This can override the organization-wide default and create a new default for that specific ad account.
  </Step>
</Steps>

## Change Creative enhancements for one batch

Use the batch-level setting when only the batch you are creating should use a different Creative enhancements setup.

<Steps>
  <Step title="Go to Launch" titleSize="h4">
    Go to `Launch`.
  </Step>
  <Step title="Open Additional settings" titleSize="h4">
    While creating a batch, open `Additional settings`.
  </Step>
  <Step title="Change Creative enhancements for the batch" titleSize="h4">
    Change the Creative enhancements settings for that batch.
  </Step>
</Steps>

Changing Creative enhancements at batch level does not change the saved organization-wide or ad account-wide defaults. It only changes the batch being created at that moment.

## Common cases

<Expandable title="I want the same Creative enhancements setting on every current and future ad account">
  Set the organization-wide default from `Settings` → `Default settings` → `Creative enhancements`.
</Expandable>

<Expandable title="I only want to change one ad account">
  Go to `Settings`, select the ad account you want to change, and update `Creative enhancements` for that ad account.
</Expandable>

<Expandable title="I only want to change one launch batch">
  Do not change the saved defaults in `Settings`. Go to `Launch`, open `Additional settings`, and adjust Creative enhancements for that batch.
</Expandable>
