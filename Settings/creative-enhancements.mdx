Use standard Creative enhancements to define the default enhancement setting Adtool should use for an ad account. Use this page when you want new launch batches to start with the same Creative enhancements setup, while still keeping the option to adjust a specific batch in `Launch`.

## What this setting affects

| **Setting** | **What it affects** |
| --- | --- |
| Selected ad account | The default Creative enhancements setting for that account. |
| `Use for all accounts` | Applies the same Creative enhancements setting across your accounts and replaces account-specific defaults. |
| `Additional Settings` in `Launch` | Lets you adjust Creative enhancements for a specific batch without changing the saved default. |

<Note>
  Creative enhancements set in `Settings` are defaults. If you change Creative enhancements inside `Launch` under `Additional Settings`, that change applies to the batch you are working on.
</Note>

## Before you save

Check these items before you save the setting:

| **Check** | **Why it matters** |
| --- | --- |
| Correct ad account selected | The setting applies to the selected account unless `Use for all accounts` is enabled. |
| `Use for all accounts` is intentional | It replaces custom Creative enhancements defaults already set on other accounts. |
| Batch-level exceptions are expected | Use `Additional Settings` in `Launch` when only one batch needs a different setup. |

## Common cases

<Expandable title="I want the same Creative enhancements setting on every account">
  Enable `Use for all accounts`, review the selected setting, and save. This applies the same default across accounts and overrides account-specific defaults.
</Expandable>

<Expandable title="I only want to change one account">
  Select the ad account, leave `Use for all accounts` off, update the Creative enhancements setting, and save.
</Expandable>

<Expandable title="I only want to change one launch batch">
  Do not change the account default in `Settings`. Go to `Launch`, open `Additional Settings`, and adjust Creative enhancements for that batch.
</Expandable>
