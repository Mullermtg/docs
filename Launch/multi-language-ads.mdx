Use multi-language ads when you want to launch the same campaign structure with localized copy and creatives for multiple languages. Adtool lets you enable language ads from the launch settings, choose the languages, and then add language-specific copy and creatives before launching.

# Step-by-step guide

<Steps>
  <Step title="Open Additional Settings" titleSize="h4">
    Go to `Launch`, then open `Additional Settings`.

    <Frame>
      ![Launch page with Additional Settings highlighted](/images/go_to_launch_additional-settings.png)
    </Frame>
  </Step>
  <Step title="Enable language ads" titleSize="h4">
    Toggle on `Language ads`, then click `Confirm Changes`.

    <Frame>
      ![Language ads toggle enabled in Additional Settings](/images/toggle_on_language_ads.png)
    </Frame>
  </Step>
  <Step title="Select languages" titleSize="h4">
    Select the languages you want to include in the launch.

    <Frame>
      ![Language selector with multiple languages selected](/images/select_languages.png)
    </Frame>
  </Step>
  <Step title="Add copy for each language" titleSize="h4">
    Write the ad copy for each individual language.

    <Frame>
      ![Multi-language copy fields for localized ad text](/images/languages_copy.png)
    </Frame>
  </Step>
  <Step title="Select creatives for each language" titleSize="h4">
    Select the creatives for the language ads, then click `Continue`.

    <Tip>
      You can select creatives for each individual language in the ad.
    </Tip>

    <Expandable title="Language matching rules">
      Adtool can automatically match creatives to languages when the file names include a supported language parameter.

      | **Language** | **Matching parameter** |
      | --- | --- |
      | Danish | `dk`, `da`, `danish`, `denmark` |
      | Swedish | `se`, `sv`, `swedish`, `sweden` |
      | Norwegian | `no`, `nb`, `norwegian`, `norway` |
      | Finnish | `fi`, `finnish`, `finland` |
      | English | `en`, `english` |
      | English (UK) | `gb`, `en`, `english`, `uk`, `britain` |
      | German | `de`, `german`, `germany` |
      | French | `fr`, `french`, `france` |
      | French (Canada) | `ca`, `fr`, `french` |
      | Spanish | `es`, `spanish`, `spain` |
      | Spanish (Mexico) | `mx`, `es`, `spanish`, `mexico` |
      | Italian | `it`, `italian`, `italy` |
      | Dutch | `nl`, `dutch`, `netherlands`, `holland` |
      | Portuguese | `pt`, `portuguese`, `portugal` |
      | Portuguese (Brazil) | `br`, `pt`, `portuguese`, `brazil` |
      | Polish | `pl`, `polish`, `poland` |
      | Czech | `cz`, `cs`, `czech`, `czechia` |
      | Japanese | `jp`, `ja`, `japanese`, `japan` |
      | Korean | `kr`, `ko`, `korean`, `korea` |
      | Chinese (Simplified) | `cn`, `zh`, `chinese`, `china` |
      | Chinese (Hong Kong) | `hk`, `zh`, `chinese`, `hong kong` |
      | Chinese (Taiwan) | `tw`, `zh`, `chinese`, `taiwan` |
      | Turkish | `tr`, `turkish`, `turkey` |
      | Russian | `ru`, `russian`, `russia` |
      | Arabic | `ar`, `arabic` |
      | Thai | `th`, `thai`, `thailand` |
      | Vietnamese | `vn`, `vi`, `vietnamese`, `vietnam` |
      | Hungarian | `hu`, `hungarian`, `hungary` |
      | Romanian | `ro`, `romanian`, `romania` |
      | Greek | `gr`, `el`, `greek`, `greece` |
      | Bulgarian | `bg`, `bulgarian`, `bulgaria` |
      | Croatian | `hr`, `croatian`, `croatia` |
      | Ukrainian | `ua`, `uk`, `ukrainian`, `ukraine` |
    </Expandable>

    <Expandable title="Examples that match and do not match">
      Examples that can match:

      - `hero_DK.jpg` \+ `hero_SE.jpg` \+ `hero_EN.jpg`
      - `banner-danish.png` \+ `banner-swedish.png`
      - `summer_campaign_da.mp4` \+ `summer_campaign_sv.mp4`
      - `DK_promo.jpg` \+ `SE_promo.jpg`
      - `ad (dk).jpg` \+ `ad (se).jpg`

      Examples that will not match:

      | **File name** | **Why it does not match** |
      | --- | --- |
      | `heroDK.jpg` | `DK` is not separated from `hero`. |
      | `hero_DK.jpg` by itself | Matching needs at least two files with the same base name. |
      | `hero_DK.jpg` \+ `hero_DK_v2.jpg` | The base names are different. |
      | `hero_DK.jpg` \+ `hero_SE.mp4` | The files mix image and video formats. |
    </Expandable>

    <Frame>
      ![Creative selection view for multi-language ads](/images/languages_select_creatives.png)
    </Frame>
  </Step>
  <Step title="Review and launch" titleSize="h4">
    Adjust the rest of the batch settings, review the language-specific copy and creatives, then launch the batch when everything is ready.
  </Step>
</Steps>
