<Warning>
  **Examples that will NOT match**

  - heroDK.jpg→ "DK" is not separated from "hero"
  - hero\_DK.jpg (alone) → needs at least 2 files with same base name
  - hero\_DK.jpg \+ hero\_DK\_v2.jpg → different base names
  - hero\_DK.jpg \+ hero\_SE.mp4 → mixed types (image\+video)
</Warning>
