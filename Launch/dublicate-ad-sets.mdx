Duplicate an ad set when you want to reuse an existing ad set setup and launch new ads into a fresh ad set.

## Duplicate an ad set

<Steps>
  <Step title="Go to 'Launch' and select the campaign" titleSize="h4">
    <img
      src="/images/Go_to_launch_select_creatives.png"
      alt="Launch page in Adtool"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Go to 'Adset' and choose the ad set to duplicate" titleSize="h4">
    <img
      src="/images/select_ad-set_dublicate.png"
      alt="Ad set duplication option in Adtool"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Adjust the new ad set settings" titleSize="h4">
    You can update:

    - **New Name:** Change the name of the new ad set.
    - **Edit Dates:** Change the start date and end date.
    - **Edit Budget:** Change the budget when the budget is set to ABO.
    - **Launch as Active:** Choose whether the new ad set launches as active or inactive.

    <img
      src="/images/copy_ad-set_settings.png"
      alt="Ad set duplication settings in Adtool"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Click 'Copy'" titleSize="h4">
    The new ad set is created with the same settings as the original, except for the changes you made during duplication.

    Adtool copies settings such as optimization goal, audience, and placement options. The new ad set does not contain ads yet, so it is ready for new ads.

    <img
      src="/images/see_copied_ad-set.png"
      alt="Copied ad set in Adtool"
      title="Copied ad set in Adtool"
      style={{ width:"100%" }}
    />
  </Step>
</Steps>
