Use carousel ads when one ad should contain multiple cards that customers can swipe through. In Adtool, carousel launches use the same creative selection flow, then a carousel-specific grouping step where you build the card order and placement versions.

# Step-by-step guide

<Steps>
  <Step title="Open Launch" titleSize="h4">
    Go to `Launch` and click `Select Creatives`.

    <Frame>
      ![Launch page with the option to select creatives for a new batch](/images/Go_to_launch.png)
    </Frame>
  </Step>
  <Step title="Select the creatives for the carousel" titleSize="h4">
    Select the creatives you want to use as carousel cards, then click `Continue`.

    <Tip>
      Use filters, `Select all`, or click and drag to select several creatives faster.
    </Tip>

    <iframe src="https://www.youtube.com/embed/Nx85-E0xQn0?autoplay=1&mute=1&loop=1&playlist=Nx85-E0xQn0&playsinline=1&controls=0&modestbranding=1&rel=0&disablekb=1&fs=0&iv_load_policy=3" title="Selecting creatives for an Adtool launch" frameBorder="0" className="w-full aspect-video rounded-xl pointer-events-none" allow="autoplay; encrypted-media" aria-hidden="true" />
  </Step>
  <Step title="Group the carousel cards" titleSize="h4">
    Adtool builds your selected creatives into carousel cards on the grouping screen. You can group them automatically with file naming or manually by hand.

    **Automatic grouping:** When your file names include a card marker such as `card 1`, `card 2`, `card 3`, Adtool groups the cards as soon as you reach the grouping screen. Each card can hold a story (`9x16`) and a feed (`1x1`, `4x5`) version, a carousel supports 2 to 10 cards, and every card must be the same media type — all images or all videos. To build several carousels in one batch, give each its own base name, like `summer_sale_card_1` and `winter_sale_card_1`.

    **Manual grouping:** If your files are not named with card markers, click `Carousel Ad` in the `Groups` panel to create a carousel group, then drag each card into it in the order it should appear. Click `Ungroup All` first if you want to redo an automatic grouping.

    For the full rules and examples, see [Carousel grouping](/Launch/carousel-grouping).

    <iframe src="https://www.youtube.com/embed/sry4CEptEwA?autoplay=1&mute=1&loop=1&playlist=sry4CEptEwA&playsinline=1&controls=0&modestbranding=1&rel=0&disablekb=1&fs=0&iv_load_policy=3" title="Grouping creatives for a carousel ad" frameBorder="0" className="w-full aspect-video rounded-xl pointer-events-none" allow="autoplay; encrypted-media" aria-hidden="true" />
  </Step>
  <Step title="Add carousel ad details" titleSize="h4">
    Click `Continue`, then fill out the remaining ad information, including primary text, headline, description, CTA, and any other required launch settings.

    To give each card its own headline, description, call to action, or destination URL, click `Edit Individually` and adjust each card.

    <iframe src="https://www.youtube.com/embed/j0WYOCZSlnU?autoplay=1&mute=1&loop=1&playlist=j0WYOCZSlnU&playsinline=1&controls=0&modestbranding=1&rel=0&disablekb=1&fs=0&iv_load_policy=3" title="Customizing carousel card text in Adtool" frameBorder="0" className="w-full aspect-video rounded-xl pointer-events-none" allow="autoplay; encrypted-media" aria-hidden="true" />
  </Step>
  <Step title="Launch the ads" titleSize="h4">
    When the batch is ready, click `Launch`.
  </Step>
</Steps>
