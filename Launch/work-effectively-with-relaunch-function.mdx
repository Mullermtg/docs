Use the relaunch function when you want to reuse a previous batch as the starting point for a new launch.

<Note>
  Relaunch reuses the ads and assets from a previous batch. Ads in Adtool are not locked to the original campaign or ad set hierarchy like they are in Meta, so you can launch the same ads into completely different campaign objectives without losing any of the assets you have created.
</Note>

## Relaunch a previous batch

<Steps>
  <Step title="Go to 'Launch'" titleSize="h4">
    <img
      src="/images/Go_to_launch.png"
      alt="Launch page in Adtool"
      title="Launch page in Adtool"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Find the batch in the batch upload history" titleSize="h4">
    Scroll to the batch upload history and click the rocket icon next to the batch you want to relaunch.

    <img
      src="/images/relaunch-function.png"
      alt="Relaunch icon in the batch upload history"
      title="Relaunch icon in the batch upload history"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Review the restored batch" titleSize="h4">
    Adtool restores the text fields, creatives, settings, and other batch details from the previous upload.

    You can adjust text, creatives, campaigns, naming, or other settings before launching again.

    <img
      src="/images/preview_of_relaunch-function.png"
      alt="Restored batch after using the relaunch function"
      title="Restored batch after using the relaunch function"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Launch the batch" titleSize="h4">
    When the adjustments are ready, launch the batch.

    <img
      src="/images/launch_relaunch_batch.png"
      alt="Launching a relaunched batch in Adtool"
      title="Launching a relaunched batch in Adtool"
      style={{ width:"100%" }}
    />
  </Step>
</Steps>
