Use Bulk Edit Naming to update ad names across a batch before launch.

## Bulk edit ad names

<Steps>
  <Step title="Go to 'Launch' and click the pencil icon" titleSize="h4">
    Click the pencil icon on the first ad in the batch.

    <img
      src="/images/got_to_launch_with_ads(1).png"
      alt="Pencil icon for editing ad names on the Launch page"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Update names with Find & Replace or Line View" titleSize="h4">
    Use Find & Replace to update multiple ad names at once, or edit names individually in Line View.

    Before using Bulk Edit Naming, make sure your default naming convention is set up in Settings.

    Use custom fields to create placeholders, such as country or funnel.

    <iframe src="https://www.youtube.com/embed/8e74xnBhvTg?autoplay=1&mute=1&loop=1&playlist=8e74xnBhvTg&playsinline=1&controls=0&modestbranding=1&rel=0&disablekb=1&fs=0&iv_load_policy=3" title="Adtool tutorial for custom fields in ad naming" frameBorder="0" className="w-full aspect-video rounded-xl pointer-events-none" allow="autoplay; encrypted-media" aria-hidden="true" />

    Shortcuts:

    - Copy text: Mac (Command \+ C) / Windows (Ctrl \+ C)
    - Paste text: Mac (Command \+ V) / Windows (Ctrl \+ V)
    - Navigate between fields: Tab
  </Step>
  <Step title="Close the pop-up when the names are ready" titleSize="h4">
    Your updated ad names are saved for the batch.
  </Step>
</Steps>
