Use 2-placement grouping when the same ad should include two creative versions for different placement groups, such as feed and Story / Reel, or feed and horizontal.

## Recommended setup

For two-placement grouping, the recommended setup is one vertical creative and one feed creative for the same ad.

- Vertical: a supported Story / Reel format, such as 9x16
- Feed: a supported square or portrait format, such as 1x1 or 4x5

Adtool can automatically group creatives when the filenames share the same base name and use different supported placement tags or dimensions. You can also group creatives manually with drag and drop.

## Group creatives before launch

  <Step title="Go to 'Launch' and select creatives" titleSize="h4">
    <img
      src="/images/Go_to_launch_select_creatives.png"
      alt="Launch page with Select Creatives highlighted"
      title="Launch page with Select Creatives highlighted"
      style={{ width:"100%" }}
    />
  </Step>
  <Step title="Select the creatives for your ads" titleSize="h4">
    Use filters, the 'select all' button, or click and drag to select the creatives you want to launch. Then click 'Continue'.

    <iframe src="https://www.youtube.com/embed/Nx85-E0xQn0?autoplay=1&mute=1&loop=1&playlist=Nx85-E0xQn0&playsinline=1&controls=0&modestbranding=1&rel=0&disablekb=1&fs=0&iv_load_policy=3" title="Selecting creatives in Adtool" frameBorder="0" className="w-full aspect-video rounded-xl pointer-events-none" allow="autoplay; encrypted-media" aria-hidden="true" />
  </Step>
  <Step title="Review the creative groups" titleSize="h4">
    If two creatives share the same base name and use different supported placement tags or dimensions, Adtool recognizes them as belonging to the same group.

    <Expandable title="Supported format and tag options">
      For the recommended two-placement setup, use one vertical creative and one feed creative.

      | **Placement** | **Accepted tags** |
      | --- | --- |
      | Story / Reel | `9x16`, `9:16`, `9_16`, `9-16`, `9.16`, `(916)`, `1080x1920`, `story`, `stories`, `reel`, `reels`, `vertical`, or a clear 9:16 pixel size like `1440x2560`. |
      | Feed placement | **Square:**`1x1`, `1:1`, `1_1`, `1-1`, `1.1`, `(11)`, `1080x1080`, `1200x1200`, `square`, `feed`, `post`, `ig`, or a clear square pixel size like `1440x1440`.<br /><br />**Portrait:**`4x5`, `4:5`, `4_5`, `4-5`, `4.5`, `(45)`, `1080x1350`, or a clear 4:5 pixel size like `1080x1350`. |

      Pixel sizes can use `x`, `X`, or `×`. For example, `1440x2560`, `1440X2560`, and `1440×2560` can all be read as pixel sizes.

      The placement tag must stand on its own in the file name. It can be separated with a space, underscore, hyphen, or brackets.

      Horizontal creatives are covered in the [3-placement grouping workflow](/Launch/grouping-with-3-placement-groups).
    </Expandable>

    Read more: [Auto-grouping guide](/Creatives/auto-grouping-guide).

    For example, 'Video V6 9x16' and 'Video V6 4x5' can be grouped together because the base name matches and the placement format changes.
  </Step>
  <Step title="Manually group creatives if needed" titleSize="h4">
    If the filenames do not match, drag and drop creatives into the correct group before continuing.
  </Step>
</Steps>
