Use 'Additional Settings' to adjust settings for the batch you are launching without changing your default settings.

## Adjust batch-level settings

<Steps>
  <Step title="Go to 'Launch' and click 'Additional Settings'" titleSize="h4">
    <Frame>
      ![Additional Settings button on the Launch page](/images/go_to_launch_additional-settings.png)
    </Frame>
  </Step>
  <Step title="Adjust the settings for this batch" titleSize="h4">
    Make the changes that should apply to the current batch.

    <Frame>
      ![Batch-level settings in Additional Settings](/images/additional_settings_upper.png)
    </Frame>
  </Step>
  <Step title="Adjust Creative Enhancement settings if needed" titleSize="h4">
    You can adjust Creative Enhancement settings for the current batch.

    If you want to add the 'Show Products' extension, choose the product set you want to use. You can also create custom 'Site Links' and save them for later use.

    These changes only apply to the batch you are launching. To change the default Creative Enhancement settings, go to 'Settings'.

    <Frame>
      ![Creative Enhancement settings for a batch](/images/additional_settings_creative-enhancement.png)
    </Frame>
  </Step>
  <Step title="Click 'Confirm Changes'" titleSize="h4">
    If 'Site Links' or 'Show Products' are turned on, they appear in the preview. If both are turned on, you can toggle between them.

    <Frame>
      ![Preview of batch-level Additional Settings changes](/images/additional_settings_preview\(1\).png)
    </Frame>
  </Step>
</Steps>
