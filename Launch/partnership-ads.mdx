Use partnership ads when you want an ad to run with a creator or partner identity. In Adtool, you can launch partnership ads in two ways: by boosting an existing creator post with a post ID or URL, or by creating a new ad from scratch with the partner as the sender.

| **Method** | **Use this when** |
| --- | --- |
| Existing post | The creator has already published an organic post, and you want to boost that post. |
| From scratch | You want to select your own creatives, write your own copy, and use the partner as the sender. |

## Launch partnership ads from existing posts

Use this method when the creator has already published an organic post and you want to boost that existing post as an ad.

<Steps>
  <Step title="Open Additional Settings" titleSize="h4">
    Go to `Launch`, then open `Additional Settings`.
  </Step>
  <Step title="Enable Partnership Ads" titleSize="h4">
    Turn on `Partnership Ads`.

    Once enabled, Adtool shows the partnership ad options for the batch.
  </Step>
  <Step title="Add the post ID or URL" titleSize="h4">
    Click `+ Add post IDs`.

    Paste the post ID or post URL into the input field, then click `Add`. Adtool will resolve the post and add it to the launch.

    <Note>
      The creator must be an approved partner before Adtool can use the post ID or URL for a partnership ad.
    </Note>
  </Step>
  <Step title="Review the resolved post" titleSize="h4">
    Confirm that Adtool resolved the correct post before continuing.

    The existing post is used as the ad content, so you do not need to select your own creative for this method.
  </Step>
  <Step title="Complete the launch details" titleSize="h4">
    Fill out any remaining required launch settings, including destination and CTA when needed.
  </Step>
  <Step title="Launch the ads" titleSize="h4">
    When the batch is ready, click `Launch`.
  </Step>
</Steps>

## Launch partnership ads from scratch

Use this method when you want to create a new ad with your own creatives and copy, while using the partner as the sender.

<Steps>
  <Step title="Open Additional Settings" titleSize="h4">
    Go to `Launch`, then open `Additional Settings`.
  </Step>
  <Step title="Enable Partnership Ads" titleSize="h4">
    Turn on `Partnership Ads`.

    Once enabled, Adtool shows the partnership ad options for the batch.
  </Step>
  <Step title="Select the partner" titleSize="h4">
    Open the partner selector and choose the partner you want to use as the sender.

    After selecting the partner, click `Continue`.
  </Step>
  <Step title="Click Select Creatives" titleSize="h4">
    Click `Select Creatives`, then select every creative you want to include in the partnership ad launch.

    You can select creatives across different placement formats. For example, if you want to launch ads with `1x1`, `4x5`, and `9x16` versions, select all of those files before continuing.

    <Tip>
      Use filters, `Select all`, or click and drag to select several creatives faster.
    </Tip>
  </Step>
  <Step title="Group matching placements" titleSize="h4">
    Adtool automatically groups creatives when their names match and the placement size is the only difference.

    If the creatives are not grouped automatically, drag them into the correct group manually before continuing. For a deeper explanation of supported placement tags and matching rules, see the [auto-grouping guide](/Creatives/auto-grouping-guide).

    <Expandable title="How grouping works for partnership ads created from scratch">
      Each group represents the creative assets that belong to the same ad. A group can contain one creative, or multiple versions of the same creative made for different placements.

      | **Creative setup** | **How to group it** |
      | --- | --- |
      | One image or one video | Keep it as a single creative group. |
      | Same creative in multiple formats | Group the matching placement versions together. |
      | Different ads in the same launch | Keep each ad in its own group. |
    </Expandable>
  </Step>
  <Step title="Add ad details" titleSize="h4">
    Click `Continue`, then fill out the remaining ad information, including primary text, headline, description, CTA, destination, and any other required launch settings.

    Review the previews for each placement before launching so you can confirm the partner identity, creative, and text look correct together.
  </Step>
  <Step title="Launch the ads" titleSize="h4">
    When the batch is ready, click `Launch`.
  </Step>
</Steps>
