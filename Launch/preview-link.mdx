Create a preview link when you want to share draft or launched ads with a client, teammate, or stakeholder.

## Create a preview link for one batch

<Steps>
  <Step title="Go to 'Launch' and create your batch" titleSize="h4">
    Create the batch as you normally would.

    ![Launch page in Adtool](/images/Go_to_launch.png)
  </Step>
  <Step title="Launch the batch or save it as a draft" titleSize="h4">
    ![Saved draft in the batch history table](/images/Save_as_draft_2.png)
  </Step>
  <Step title="Copy the preview link" titleSize="h4">
    Scroll to the batch history and click the preview link icon. Adtool copies the preview link for the batch.

    ![Preview link icon in the batch history](/images/copy_preview_link.png)
  </Step>
  <Step title="Share or open the link" titleSize="h4">
    The preview shows all ads from the batch. You can view the preview in feed or reel format.

    If the batch includes multiple text variations, each ad includes a 'Show all variations' option.

    ![Preview page showing ads from a batch](/images/copy_preview_link\(1\).png)
  </Step>
</Steps>

## Create one preview link for multiple batches

Select the batches you want to include from your batch history, then click the preview link icon. Adtool copies one link that shows all selected batches in the same preview.

Use this when you want to share multiple draft or launched batches without sending separate preview links.
