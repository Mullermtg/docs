### Create a preview link for one batch

<Steps>
  <Step title="Go to 'Launch' and create your batch">
    Simply create the batch as you normally would.

    ![Go To Launch](/images/Go_to_launch.png)
  </Step>
  <Step title="Either launch or save the batch as draft">
    ![Save As Draft 2](/images/Save_as_draft_2.png)
  </Step>
  <Step title="Scroll down to the batch history and click on the preview link icon">
    This will copy the preview link for the batch.

    ![Copy Preview Link](/images/copy_preview_link.png)
  </Step>
  <Step title="Send the link to the desired person or paste it in a new tab">
    This preview will show all the ads from the batch.

    You are able to watch the preview in feed or reel format. Below each ad you'll see a 'Show all variations' option if you've made multiple text variations.

    ![Copy Preview Link(1)](/images/copy_preview_link\(1\).png)
  </Step>
</Steps>

### Create one preview link for multiple batches

You can create one preview link that includes several batches from your batch history.

To do this, select the batches you want to include, then click the preview link icon. Adtool will copy one link that shows all selected batches in the same preview.

Use this when you want to share multiple draft or launched batches with a client, teammate, or stakeholder without sending separate preview links for each batch.
